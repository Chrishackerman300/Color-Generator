Sean bienvenido a este proyecto. 

Estaré haciendo un generador de color (color generator) con HTML, CSS y JavaScript,
donde habrá un color hexadecimal predeterminado que será visible en el display de la web.

Tendrá un boton para cambiar de color hexadecimal, donde cada vez que el usuario le de clic al boton y este va a cambiar de código hexadecimal y cambiará el color del display de la web.
